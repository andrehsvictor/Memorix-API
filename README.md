# Features

## Authentication

- Login (obtain JWT token)
- Logout (invalidate JWT token)
- Refresh token (obtain new JWT token)
- Verify e-mail address (send verification e-mail)
- Reset password (send reset password e-mail)

## User

- Create user (sign up)
- Update your user information
- Delete your user account
- Get your user information
- Update your password

## Deck

- Create deck
- Update deck
- Delete deck
- Get deck information by slug
- Get all your decks
- Bulk delete decks

## Card

- Create card
- Update card
- Delete card
- Get card information by id
- Get all cards
- Get all cards by deck slug
- Bulk delete cards

## Image

- Upload image
- Get image by name
